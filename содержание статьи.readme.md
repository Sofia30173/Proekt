# Выявление сердечно-сосудистых заболеваний с помощью Python

## Введение
Проект исследует применение Python для предсказания сердечно-сосудистых заболеваний (ССЗ) на основе медицинской истории пациентов, включая данные о уровне сахара, артериальном давлении, боли в груди, возрасте и поле. Python, как высокоуровневый и объектно-ориентированный язык программирования, демонстрирует свою эффективность при использовании алгоритмов машинного обучения, достигая точности около 83% с помощью классификатора случайного леса.

## Методы исследования
Используются различные алгоритмы машинного обучения, такие как K-ближайший сосед, деревья решений, поддерживающие векторные машины (SVM) и случайный лес. Алгоритм случайного леса строит множество деревьев решений и объединяет их результаты для определения конечной классификации. Другие алгоритмы, такие как логистическая регрессия, предоставляют дополнительные подходы для анализа данных и прогнозирования.

## Результаты анализа
Проект подтверждает преимущества Python и его библиотек (Matplotlib, NumPy, Pandas), предлагая высокую производительность в обработке и интерпретации данных. Алгоритм случайного леса, включая различные методы обработки, обеспечил значительные улучшения в прогнозировании ССЗ, что позволяет врачам лучше использовать данные для принятия решений. 

## Заключение
Применение Python в кардиологии способствует эффективному анализу и предсказанию сердечно-сосудистых заболеваний. Модель прогнозирования, основанная на обширных данных и машинном обучении, предлагает врачу более точные и обоснованные результаты, что улучшает систему предоставления медицинских услуг.
